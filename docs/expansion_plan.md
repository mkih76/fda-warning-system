> 版本：v1.0 | 日期：2026-04-29 | 作者：Hermes Agent

---

## 一、扩展目标

从单一的"FDA警告信监控平台"升级为**FDA合规知识平台**，覆盖制药、化妆品、食品三大领域，兼顾宏观趋势与微观GMP细则，为合规从业者提供一站式信息服务平台。

### 核心定位

```
当前：FDA Warning Letter Intelligence Platform
  ↓
升级：FDA Compliance Intelligence Platform
    (美国FDA合规情报平台)
```

## 三、新增项目规划

### 项目1：FDA化妆品合规情报站

**背景**：MoCRA（2022年化妆品现代化法案）2025-2026年全面生效

| 项目 | 内容 |
|------|------|
| **数据采集** | FDA化妆品设施注册、产品清单、不良反应报告 |
| **功能** | 成分合规检查、MoCRA时间线追踪、法规更新推送 |
| **差异化** | 中文解读MoCRA、成分限制速查表 |
| **目标用户** | 中国化妆品出口企业、跨境卖家 |

**关键内容**：
- FDA化妆品设施注册数据库查询
- 26种需报告的过敏原成分清单
- 滑石粉石棉检测要求
- PFAS限制（加州/科罗拉多等州法）

### 项目3：GMP知识库（中文百科）

**定位**：GMP合规领域的"维基百科"

| 模块 | 内容 | 形式 |
|------|------|------|
| **条款解读** | 21 CFR 211全条款中文解读 | 图文 |
| **案例库** | 100+警告信案例分析 | 图文/视频 |
| **自查工具** | 在线自查问卷 | 交互式 |
| **术语词典** | GMP专业术语中英对照 | 搜索 |
| **审核模拟** | FDA审核常见问题 | Q&A |

**内容示例**：

```
21 CFR 211.68 自动化系统要求
━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📌 原文要点：
  任何在GMP操作中使用的计算机化系统必须经过验证。

🔍 通俗解释：
  如果你用电脑控制生产设备、记录数据、管理配方，
  FDA要求你证明这个系统是可靠的，不会出错。

⚠️ 常见违规：
  - 没有做计算机化系统验证（CSV）
  - 审计追踪功能关闭
  - 用户权限管理混乱

✅ 如何合规：
  1. 对所有GMP相关软件做IQ/OQ/PQ验证
  2. 确保审计追踪功能开启且不可关闭
  3. 建立用户权限管理制度
  4. 定期进行系统再验证

📋 案例：某药企因审计追踪被关闭收到警告信
```

## 四、法律风险规避策略

### 4.1 核心原则

```
我们提供：信息聚合、翻译解读、趋势分析
我们不提供：法律建议、合规咨询、具体操作指导
```

### 4.2 免责声明设计

在每个页面底部和所有内容中加入：

```
⚕️ 免责声明：
本平台提供的信息仅供参考，不构成法律建议或合规指导。
具体合规问题请咨询专业法律顾问。
FDA法规可能随时变更，请以FDA官方发布为准。
```

### 4.3 风险分级

| 风险等级 | 内容类型 | 处理方式 |
|---------|---------|---------|
| 🟢 低 | 警告信原文翻译、数据统计 | 直接发布 |
| 🟡 中 | GMP条款解读、案例分析 | 加免责声明+引用原文 |
| 🔴 高 | 具体合规建议、操作指导 | **不做** 或引用FDA官方指南 |

### 4.4 内容审核机制

| 步骤 | 内容 | 责任人 |
|------|------|--------|
| 1. 原始采集 | FDA原文/数据 | 自动 |
| 2. AI翻译 | 初步翻译 | AI |
| 3. 人工审核 | 专业术语准确性 | 合规专家 |
| 4. 法律审核 | 是否涉及建议性内容 | 法律顾问 |
| 5. 发布 | 带免责声明 | 编辑 |

## 十、法律合规保障

### 10.1 必做清单

- [ ] 每个页面添加免责声明
- [ ] 内容分类：信息类 vs 建议类
- [ ] 建议类内容仅引用FDA官方指南
- [ ] 咨询服务由合作律所提供
- [ ] 用户协议明确平台角色
- [ ] 保留FDA原文链接作为数据来源

### 10.2 禁止内容

- ❌ 具体的合规操作建议
- ❌ 对特定企业的合规评价
- ❌ 法律意见
- ❌ 替代FDA官方指南的解读

### 10.3 推荐合作

| 合作方 | 作用 |
|--------|------|
| 医药律所 | 内容审核 + 法律咨询 |
| GMP培训师 | 内容创作 + 专业背书 |
| 行业协会 | 渠道推广 + 品牌背书 |
