## 1. 问题描述 (Problem Description)

### 1.1 翻译内容问题

| 编号 | 问题类型 | 影响范围 | 严重程度 |
|------|----------|----------|----------|
| P-1 | 翻译中包含网页导航残留（"跳至主要内容"、"首页"、"返回顶部"等） | 223/258 (86.4%) | 🔴 高 |
| P-2 | 翻译中包含FDA网站页脚模板文字（社交媒体链接、联系电话等） | 113/258 (43.8%) | 🟡 中 |
| P-3 | 翻译中出现AI自行添加的总结性词汇（"总之"、"综上所述"、"值得注意的是"等） | 31/258 (12.0%) | 🔴 高 |
| P-4 | 翻译未严格按原文1:1翻译，存在概括、省略、自我总结现象 | 部分案例 | 🔴 高 |
| P-5 | 少数翻译英文比例过高或长度异常过短 | 11/258 (4.2%) | 🟡 中 |

### 1.2 排版格式问题

| 编号 | 问题类型 | 说明 |
|------|----------|------|
| P-6 | 翻译未采用FDA官网排版格式 | 字体、字号、行高、标题样式均与FDA官网不一致 |
| P-7 | 中英文框体不一致 | 中文翻译框与英文原文框样式不统一 |
| P-8 | 段落结构丢失 | 原文段落层次在翻译中未保留 |

### 问题根源 (Root Cause)

1. **原文未预处理：** 抓取的原文包含FDA网页的导航栏、页脚、侧边栏等非正文内容
2. **翻译Prompt缺陷：** 未明确要求"严格逐句翻译，不得添加总结、概括或省略"
3. **排版未对标FDA官网：** 翻译输出未按FDA官方样式进行渲染
4. **无翻译质量校验：** 翻译完成后未进行质量检查

## 2. FDA官网排版规范 (Typography Spec)

基于对 https://www.fda.gov 警告信页面的实测分析：

### 2.1 字体系统

| 元素 | 字体 | 字号 | 行高 | 字重 | 颜色 |
|------|------|------|------|------|------|
| 页面标题 (h1) | Merriweather, serif | 36px (2.25rem) | 45px (1.25) | 700 Bold | #333 |
| 节标题 (strong) | Helvetica, Arial, sans-serif | 18px (1.125rem) | 29.16px (1.62) | 700 Bold | #333 |
| 正文段落 (p) | Helvetica, Arial, sans-serif | 18px (1.125rem) | 29.16px (1.62) | 400 Regular | #333 |
| 地址/元信息 | Helvetica, Arial, sans-serif | 16px (1rem) | 25.92px (1.62) | 400 Regular | #333 |
| 小标签 | Helvetica, Arial, sans-serif | 14px (0.875rem) | 20px | 400 Regular | #333 |

### 2.2 间距系统

| 元素 | margin-bottom | margin-top | 说明 |
|------|---------------|------------|------|
| 正文段落 | 16px | 0 | 段落间距 |
| 标题前 | - | - | bold标题嵌入段落中 |
| 分隔线 (hr) | 16px | 16px | 节分隔 |
| 页面标题 | 15px | - | h1底部间距 |

### 2.3 布局

| 属性 | 值 | 说明 |
|------|------|------|
| 背景色 | #ffffff | 纯白背景 |
| 文字色 | #333333 | 深灰文字 |
| 最大宽度 | ~750px | 正文区域 |
| 内边距 | 15px | 内容区左右 |

### 2.4 警告信标准结构

```
[日期]

**WARNING LETTER**          ← bold, 节标题

Dear Mr./Ms. [Name]:        ← 正文

[正文段落...]                ← 正文段落, 18px, 行高1.62

**[节标题]**                 ← bold, 18px, 作为段落分隔

[正文段落...]

---

Sincerely,

[签名]
[职务]
```

### 2.5 等比缩放规则

由于FDA原文18px在我们的详情页卡片中偏大，采用 **0.889 比例**（18px → 16px）：

| FDA官网尺寸 | 我们页面尺寸 | 说明 |
|-------------|-------------|------|
| 36px (h1) | 24px | 标题，已在详情页顶部处理 |
| 18px (正文) | 16px | 正文段落 |
| 29.16px (行高) | 25.92px | 行高 1.62 |
| 16px (margin) | 14px | 段落间距 |
| 16px (元信息) | 14px | 地址等 |
| 14px (小标签) | 12px | 辅助信息 |

## 3. 纠正措施 (Corrective Action)

### CA-1：原文预处理 — 清理非正文内容

**脚本：** `/root/fda-warning-system/scripts/clean_fulltext.py`

**清理规则：**
- ✅ 移除：`跳至主要内容`、`跳至FDA搜索`、`跳至本节菜单`、`跳至页脚链接`
- ✅ 移除：`在Facebook上关注FDA`、`在X上关注FDA`、`在Instagram上关注FDA`
- ✅ 移除：`1-888-INFO-FDA`、`返回顶部`、`联系电话`
- ✅ 移除：`反馈 联系FDA`、`订阅FDA RSS源`、`返回FDA首页`
- ✅ 移除：`首页`、`主页`、`检查、合规、执法和刑事调查`等面包屑导航
- ✅ 保留：警告信正文（日期、收件人、正文内容、法律引用、签名）

### CA-2：翻译Prompt优化 — 严格1:1翻译

**新Prompt：**
```
你是一名专业的FDA法规文件翻译员。请严格按以下要求将FDA警告信翻译为中文：

【核心原则】
1. 逐句翻译：每一句英文原文都必须翻译为对应中文，不得遗漏
2. 禁止添加：不得添加原文没有的内容，包括：总结、概括、评价、注释、补充说明
3. 禁止省略：不得省略任何原文内容，包括法律条文引用、地址、日期、编号
4. 保留格式：保留原文的段落结构、换行位置、缩进层次
5. 保留专有名词：FDA、CGMP、21 CFR、FD&C Act等法规术语保留英文原文
6. 保留原文英文：人名、公司名、街道地址保留英文原文
7. 仅输出翻译：只输出翻译结果，不输出任何额外说明、标签或格式标记

【禁止出现的词汇】
- "总之"、"综上所述"、"概括来说"、"简而言之"
- "值得注意的是"、"需要指出"、"关键要点"、"主要发现"
- 任何原文中没有的总结性或评论性语句

请翻译以下FDA警告信正文：
```

### CA-3：排版输出 — 对标FDA官网样式

**在 `formatLetterContent()` 函数中实现：**

翻译输出的HTML格式化规则：
1. **段落检测：** 双换行符分割为 `<p>` 标签
2. **标题检测：** bold标记的文本或全大写行 → `<h3>` 或 `<strong>`
3. **日期行：** 开头的日期行（如 "January 06, 2022"）→ 独立段落
4. **称呼行：** "Dear Mr./Ms." 开头 → 独立段落
5. **签名区：** "Sincerely," 后的内容 → 独立区域
6. **法律引用：** "21 U.S.C."、"21 CFR" 等 → 蓝色高亮
7. **分隔线：** "---" 或连续横线 → `<hr>` 标签

**CSS样式对标FDA：**

```css
.letter-content {
  font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Helvetica, Arial, sans-serif;
  font-size: 16px;           /* FDA 18px × 0.889 */
  line-height: 1.62;         /* FDA 标准行高 */
  color: #333333;            /* FDA 标准文字色 */
  max-width: 750px;          /* FDA 正文宽度 */
}

.letter-content .letter-heading {
  font-size: 16px;
  font-weight: 700;          /* FDA bold标题 */
  margin: 24px 0 14px 0;
  padding-bottom: 8px;
  border-bottom: 1px solid #e5e7eb;
}

.letter-content .letter-para {
  margin-bottom: 14px;       /* FDA 16px × 0.889 */
  text-indent: 0;            /* FDA 无首行缩进 */
}

.letter-content .letter-date {
  font-size: 14px;
  color: #666;
}

.letter-content .letter-salutation {
  font-style: italic;
}

.letter-content .cfr-inline {
  color: #2563eb;
  font-weight: 600;
  background: #eff6ff;
  padding: 0 2px;
  border-radius: 2px;
}
```

## 4. 预防措施 (Preventive Action)

### PA-1：翻译质量自动校验

**校验脚本：** `/root/fda-warning-system/scripts/validate_translation.py`

| 检查项 | 权重 | 通过条件 |
|--------|------|----------|
| 长度比 | 30% | 0.3 ≤ len(zh)/len(en) ≤ 0.8 |
| 无AI总结词 | 25% | 不包含"总之"等总结性词汇 |
| 无导航残留 | 25% | 不包含"跳至"等网页模板文字 |
| 段落一致性 | 20% | 中英段落数差异 ≤ 20% |

### PA-2：排版一致性检查

- 翻译输出必须通过 `formatLetterContent()` 格式化
- 检查HTML结构是否包含 `<p>` 和 `<h3>` 标签
- 验证段落数量与原文大致一致

### PA-3：定期抽检机制

- 每批翻译完成后，随机抽取10%进行人工抽检
- 抽检结果反馈到Prompt优化和校验规则中
