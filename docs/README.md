## 项目概述

**项目名称**：FDA 警告信智能监控平台

**项目定位**：面向国内制药、化妆品、食品行业从业人员的一站式合规信息平台

**目标用户**：
- 质量管理人员
- 法规事务专员
- 研发人员
- 供应链管理人员

## 技术架构

- **框架**：Vue 3 + Vite

- **数据解析**：BeautifulSoup

- **Web服务器**：Caddy
- **反向代理**：Caddy
- **域名**：fda.19990419.top

## 数据安全

### FDA数据保存策略

```sql
-- 数据库备份
sqlite3 fda_warning.db ".backup 'fda_warning_backup.db'"

-- 定时备份（每日凌晨3点）
0 3 * * * sqlite3 /root/data/fda_warning.db ".backup /root/backups/fda_$(date +\%Y\%m\%d).db"

-- 保留最近30天备份
find /root/backups -name "fda_*.db" -mtime +30 -delete
```

### 数据导出功能

```python
# 支持导出格式

- PDF：适用于打印存档
```
