# 🚀 FDA 警告信系统 - 超简单启动指南

## 方法一：一键启动（推荐）👍

### Windows 用户：

1. **双击运行** `start.bat` 文件
2. 等待自动安装和启动（首次需要几分钟）
3. 浏览器会自动打开网站

就这样！完成了！🎉

### Mac/Linux 用户：

1. **打开终端**（Terminal）
2. **进入项目目录**：
   ```bash
   cd /path/to/fda-warning-system
   ```
3. **运行启动脚本**：
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
4. 等待自动安装和启动
5. 浏览器会自动打开网站

## 方法二：手动启动（如果一键启动失败）

### 第一步：启动后端

**Windows 用户：**

1. **打开第一个命令行窗口**
2. **复制粘贴以下命令**（一行一行执行）：
   ```bash
   cd C:\Users\22975\fda-warning-system\backend
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   uvicorn app.main:app --reload --port 8790
   ```

3. **等待看到这个**：
   ```
   INFO:     Uvicorn running on http://0.0.0.0:8790
   ```
   保持这个窗口打开！不要关闭！

**Mac/Linux 用户：**

1. **打开终端**
2. **复制粘贴以下命令**：
   ```bash
   cd ~/fda-warning-system/backend
   python3 -m venv .venv
   source .venv/bin/activate
   pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
   uvicorn app.main:app --reload --port 8790
   ```

---

### 第二步：启动前端

**打开第二个命令行窗口（Windows）或新终端窗口（Mac/Linux）**

**Windows 用户：**
```bash
cd C:\Users\22975\fda-warning-system\frontend
npm install
npm run dev
```

**Mac/Linux 用户：**
```bash
cd ~/fda-warning-system/frontend
npm install
npm run dev
```

**等待看到这个**：
```
VITE v5.x.x  ready in XXX ms

➜  Local:   http://localhost:5173/
```

---

### 第三步：打开网站

**打开浏览器，访问**：http://localhost:5173

完成！✅

## ❓ 常见问题

### 问题 1：提示"python 不是内部或外部命令"

**原因**：没有安装 Python 或没有添加到 PATH

**解决**：
1. 下载安装 Python：https://www.python.org/downloads/
2. 安装时勾选 ✅ "Add Python to PATH"
3. 重新运行

---

### 问题 2：提示"node 不是内部或外部命令"

**原因**：没有安装 Node.js

**解决**：
1. 下载安装 Node.js：https://nodejs.org/
2. 选择 LTS 版本
3. 重新运行

---

### 问题 3：pip 安装很慢

**原因**：网络问题

**解决**：
使用国内镜像（启动脚本已自动配置）：
```bash
pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

---

### 问题 4：npm install 很慢

**原因**：网络问题

**解决**：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

---

### 问题 5：端口被占用

**提示信息**：`Address already in use`

**解决**：
1. 关闭占用端口的程序
2. 或者使用其他端口：
   ```bash
   # 后端使用 8791 端口
   uvicorn app.main:app --reload --port 8791
   ```

---

### 问题 6：网站打不开

**检查清单**：
- [ ] 后端窗口是否还在运行？（不要关闭！）
- [ ] 前端窗口是否还在运行？（不要关闭！）
- [ ] 访问地址是否正确？http://localhost:5173

## 🛑 如何停止服务

### 如果使用一键启动脚本：

**Windows**：
- 关闭弹出的两个命令行窗口

**Mac/Linux**：
- 在终端按 `Ctrl + C`

### 如果手动启动：

- 关闭所有打开的命令行/终端窗口
- 或者在每个窗口按 `Ctrl + C`

## 📞 还是不行？

### 查看错误信息：

1. **后端错误**：查看第一个命令行窗口
2. **前端错误**：查看第二个命令行窗口
3. **浏览器错误**：按 F12 打开开发者工具，查看 Console

### 常用调试命令：

```bash
# 检查 Python
python --version

# 检查 Node.js
node --version

# 检查端口占用
# Windows:
netstat -ano | findstr :8790
netstat -ano | findstr :5173

# Mac/Linux:
lsof -i :8790
lsof -i :5173
```
