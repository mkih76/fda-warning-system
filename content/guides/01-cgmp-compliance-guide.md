# CGMP合规入门指南

## 概述

本指南为制药、化妆品和食品行业的新手提供CGMP（现行药品生产质量管理规范）的基础知识和实施步骤，帮助企业建立符合FDA要求的质量管理体系。

## 适用范围

- **适用行业**：制药、化妆品、食品
- **适用企业**：新成立企业、小型企业、合规起步阶段企业
- **适用阶段**：质量体系建立初期

## 核心要求

### 1. 人员要求

**法规依据**：21 CFR 211.25

**具体要求**：
1. 配备足够的合格人员
2. 建立培训程序
3. 保存培训记录

**实施步骤**：
1. 确定关键岗位人员
2. 制定岗位职责说明书
3. 建立培训计划
4. 实施培训并记录
5. 评估培训效果

**检查要点**：
- 是否有组织架构图
- 是否有岗位职责说明书
- 是否有培训计划和记录
- 培训记录是否完整

### 2. 厂房设施要求

**法规依据**：21 CFR 211.42

**具体要求**：
1. 厂房设计适当
2. 区域划分合理
3. 环境监控有效

**实施步骤**：
1. 进行厂房设施设计
2. 建立区域划分标准
3. 实施环境监控
4. 定期维护保养

**检查要点**：
- 厂房布局是否合理
- 是否有区域划分图
- 环境监控记录是否完整
- 维护保养记录是否齐全

### 3. 设备要求

**法规依据**：21 CFR 211.63, 21 CFR 211.68

**具体要求**：
1. 设备设计适当
2. 设备得到适当维护
3. 设备经过适当校准

**实施步骤**：
1. 选择适当的设备
2. 建立设备台账
3. 制定维护保养计划
4. 建立校准程序
5. 实施维护和校准

**检查要点**：
- 是否有设备清单
- 是否有维护保养计划
- 是否有校准程序
- 维护和校准记录是否完整

### 4. 生产过程控制要求

**法规依据**：21 CFR 211.100

**具体要求**：
1. 工艺经过验证
2. 关键参数受控
3. 批记录完整

**实施步骤**：
1. 开展工艺验证
2. 确定关键质量属性
3. 建立过程控制程序
4. 设计批记录模板
5. 实施过程监控

**检查要点**：
- 是否有工艺验证报告
- 是否有过程控制程序
- 批记录是否完整准确
- 偏差处理是否有效

### 5. 质量控制要求

**法规依据**：21 CFR 211.160, 21 CFR 211.192

**具体要求**：
1. 检验方法经过验证
2. 检验记录完整
3. OOS调查有效

**实施步骤**：
1. 建立检验标准操作程序
2. 开展方法验证
3. 设计检验记录模板
4. 建立OOS调查程序
5. 实施质量放行

**检查要点**：
- 是否有检验SOP
- 是否有方法验证报告
- 检验记录是否完整
- OOS调查是否有效

### 6. 文件记录要求

**法规依据**：21 CFR 211.180, 21 CFR 211.186, 21 CFR 211.188

**具体要求**：
1. 文件管理程序完善
2. 批生产记录完整
3. 批检验记录完整

**实施步骤**：
1. 建立文件管理程序
2. 设计批记录模板
3. 建立记录保存程序
4. 实施记录审核

**检查要点**：
- 是否有文件管理程序
- 批记录是否完整
- 记录保存是否符合要求
- 记录审核是否有效

## 常见问题

### Q1：CGMP和GMP有什么区别？

**A**：CGMP中的"C"代表"Current"，强调企业应采用当前的技术和最佳实践来满足法规要求。GMP是基础要求，CGMP是动态的、持续改进的要求。

### Q2：小型企业如何启动CGMP合规？

**A**：
1. **从小处着手**：先建立核心文件和程序
2. **利用资源**：参考FDA指南和行业最佳实践
3. **寻求帮助**：考虑聘请合规顾问
4. **分阶段实施**：不要试图一次性完成所有工作

### Q3：如何应对FDA检查？

**A**：
1. **日常准备**：保持文件记录完整、现场整洁
2. **检查前准备**：组织预检查、准备文件
3. **检查中配合**：如实回答问题、提供所需文件
4. **检查后跟进**：及时回复FDA、实施CAPA

## 实施案例

### 案例1：某小型制药企业

- **背景**：新成立的制药企业，首次申请FDA检查
- **挑战**：资源有限，缺乏合规经验
- **解决方案**：
  1. 聘请合规顾问指导
  2. 分阶段建立文件体系
  3. 重点培训关键岗位人员
- **结果**：成功通过FDA检查，获得批准
